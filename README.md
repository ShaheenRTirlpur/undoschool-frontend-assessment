
# UndoSchool Course Browse Assessment

## 🚀 Project Overview
This is a responsive course browsing page built using React + Vite + Tailwind CSS.

Users can:
- Search courses
- Filter by category
- View course cards in grid layout
- Responsive design for all devices

---

## 🛠 Tech Stack
- React
- Vite
- Tailwind CSS
